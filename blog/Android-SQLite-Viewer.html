Simple Android SQLite Viewer

SQLite is an integrated database solution for Android Applications. But, working with them might be difficult. Even more difficult is debugging and checking if the desired data is inserted or updated correctly.

There are several solutions available to view the Data from the Database files. But, almost all of them are complex. That’s when I stumbled upon this really simple solution.

Just add the following code to your build.gradle file and run your app in debug mode. Then you can point your browser to the "ip-address-of-the-device":8080 to view and edit your SQLite databases.

If you are using an emulator to test your application, then you need to forward the tcp 8080 port to your computer's 8080 port.

To do that open terminal/cmd and navigate to the path where your Android SDK is located. Then navigate to the dir 'platform-tools'. Under Android Studio > Tools > SDK Manager.
From there, run the following command to forward the tcp port. adb forward tcp:8080 tcp:8080 THen for emulators, you need to point your browser to localhost:8080
