GitHub for Unity | Bring Git and Github into Unity

The GitHub for Unity extension brings Git and GitHub into Unity, integrating source control into your work with friendly and accessible tools and workflows.

GitHub for Unity is a amazing tool for all the studios/developers that make use of other third-party tools to manage their Git Repo with Unity. With this, we don't need to navigate away from Unity IDE and also, it will better handle files with Git LFS.

I am yet to test the Asset in depth but, as far as I have tried, it seems to perfom better than other VCS.
